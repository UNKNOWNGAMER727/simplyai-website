# SimplyAI — Installer Field Guide

> **For the person doing installs.** Read this before your first job. Keep it handy on your phone or laptop.
> Goal: in and out in 30–60 minutes. Install, confirm it works, leave.

---

## 1. What is Perplexity AI?

Perplexity AI is an AI-powered search and research tool that answers questions in plain English — like having a brilliant assistant who can instantly research anything, summarize documents, and draft emails. Instead of Googling and clicking through ten links, you ask Perplexity once and get a direct, sourced answer in seconds. Businesses use it to save hours every week on research, writing, and answering repetitive questions. Your job is to get it set up so their team can start using it immediately.

## 2. What Your Job Is

**Show up. Install. Leave.**

You are not a teacher. You are not a consultant. You are a technician.

- You arrive at the client's location

## 3. Step-by-Step Install Process

### On-Site: Install Perplexity on **Mac** (Step-by-Step)

1. Open **Safari** or **Chrome** on the client's Mac
2. Go to `perplexity.ai`
3. Click **Sign Up**
4. Use the client's email address to create the account
5. Check their email for the verification code — enter it
6. Confirm the Perplexity homepage loads and the search bar works
7. **Pin to Dock:**
   - Open **Safari**: In the menu bar → File → Add to Dock (macOS Sonoma+)
   - Open **Chrome**: Click the three-dot menu → Cast, save, and share → Create shortcut → check "Open as window" → Add
   - Drag the shortcut to the Dock if not already there
8. If they use Chrome: install the **Perplexity Chrome Extension**
   - Go to `chromewebstore.google.com` → search "Perplexity" → Install
9. Type a test question (e.g., *"What is Perplexity AI?"*) — confirm a response appears
10. Bookmark `perplexity.ai` (`Cmd+D`)

---

### On-Site: Install Perplexity on **Windows** (Step-by-Step)

1. Open **Chrome** or **Edge** on the client's Windows PC
2. Go to `perplexity.ai`
3. Click **Sign Up**
4. Use the client's email address to create the account
5. Check their email for the verification code — enter it
6. Confirm the Perplexity homepage loads
7. **Pin to Taskbar:**
   - **Chrome**: Click three-dot menu → Cast, save, and share → Create shortcut → check "Open as window" → Create → right-click the icon in taskbar → Pin to taskbar
   - **Edge**: Click three-dot menu → Apps → Install this site as an app → Install → right-click in taskbar → Pin to taskbar
8. Install the **Perplexity Chrome Extension** (if using Chrome):
   - Go to `chromewebstore.google.com` → search "Perplexity" → Add to Chrome
9. Type a test question to confirm it works
10. Bookmark `perplexity.ai` (`Ctrl+D`)

## 8. Common Issues + Fixes

### "I'm not getting the verification email"
- Check spam/junk folder
- Try resending the verification from the Perplexity sign-up page
- If Gmail: go to `gmail.com` in the browser directly
- If still nothing: create account with **Google Sign-In** instead of email

### "I already have a Perplexity account"
- Just log in. Don't create a duplicate.
- If they forgot their password: click **Forgot password** on the sign-in page

### "The extension won't install"
- Make sure Chrome/Edge is up to date: `chrome://settings/help`
- Try in a private/incognito window first to rule out extensions conflict
- Skip the extension if it keeps failing — the website version is fully functional

### "Perplexity loads slow / doesn't respond"
- Check internet speed — try `fast.com` in a browser tab
- If their network is slow, switch to your hotspot temporarily for the setup
- Reload the page

### "Company computer blocks installs / website is blocked"
- **Stop immediately** and text Kyle
- Some corporate IT policies block AI tools — this is above your pay grade
- Do not try to work around IT restrictions

### "The computer is really old / runs slow"
- Perplexity is browser-based — it should still work
- If the browser is old (IE or ancient Chrome), update it first: go to `google.com/chrome` and install the latest version
- If it's truly unusable, text Kyle before giving up

### "They want me to set up more devices than the booking says"
- Text Kyle first
- Do not add extra devices without Kyle's approval — there may be an upsell involved

### "Client isn't home / business is closed when you arrive"
- Take a photo of the location (timestamp proof)
- Text Kyle immediately
- Do not wait more than 15 minutes
